The application is set to run in a virtual environment, and requires various packages to be installed in order to run.  The steps below are set up to help facilitate running the application

1. Activate the virtual environment
  -In the command line, navigate to the folder where the application was downloaded. 
  -Enter the following text into the command line to activate the virtual environment
	
	project_env\Scripts\activate.bat

  -You should now see the text '(project_env)' at the beginning of the line


2. Install requirements necessary to the run application 
  - Next enter the following text to install the packages required for this application

	pip install -r requirements.txt 

3. Launch Application
  - With everything installed, the application is ready to launch.
  - enter the following text to lauch the application

	Python FilteredBrews.py
